How to
Cancel an Order

Step-by-step guide for canceling a patient order

01
Go to the Dashboard
Navigate

Log in to the admin portal. You will land on the Home Dashboard.

02
Open Orders → Orders
Navigate

In the left sidebar under Management, click Orders to expand it, then select Orders from the submenu.

03
Find & Click the Patient's Name
Select

Browse the orders list and click on the patient's name (shown in blue) to open their order details.

04
Set Order Status to "Canceled"
Cancel

Scroll right to find the Order Status dropdown on the patient's order row. Click the dropdown and select Canceled from the list.

⚠️
Important: Selecting Canceled will stop all further processing of this order. This action cannot be undone. Make sure you are canceling the correct order before proceeding.
💾
Don't forget to click Save on the order row after selecting Canceled to confirm the status change.
🚫
Do not delete a patient. Never use the delete option — always use the Canceled status instead. Deleting a patient is permanent and cannot be recovered.
✅ Order Successfully Canceled
