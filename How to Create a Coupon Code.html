How to Create a
Coupon Code

Step-by-step guide for creating discount coupon codes

01
Go to the Dashboard
Navigate

Log in to the admin portal. You will land on the Home Dashboard.

02
Open Discounts → Coupon Codes
Navigate

In the left sidebar under Sales & Channels, click Discounts to expand it, then click Coupon Codes.

03
Click "+ Add New"
Action

The Coupon Codes page shows all existing coupons. Click the + Add new button in the top right to create a new one.

04
Fill in the Coupon Details
Configure

Fill in the coupon form with a Name, Code, Expiration, Discount Type, Amount, and optional Max. Threshold.

💲
Discount Type: You can set the code as a Percentage (e.g. 10% off) or a Fixed Amount (e.g. $50 off). Select the appropriate type from the Discount Type dropdown.
👤
Eligible Patients: You can set the code to apply to All Patients or restrict it to Specific Patients only. Choose the appropriate option under the Eligible Patients section.
Discount Type options
Eligible Patients
05
Set Usage & Redemption Rules
Configure

Under Usage, select the coupon Type (One Time, First Order, All Orders, N° of Orders, First Payment Only) and set Redemptions to Infinite or Limited.

Usage Type
Redemptions
06
Select Products & Save
Finish

Under Products application, choose which products the coupon applies to — you can select specific drugs or apply it to all. Then click Save to create the coupon.

Select Products
Save Coupon
💡
Pro Tip: Use a clear naming convention for your codes — e.g. FIRSTNAME50 for a $50 patient-specific discount. Always set the correct Type (One Time vs All Orders) to avoid unintended repeated use.
✅ Coupon Code Created!
